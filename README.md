# 🌊 Ocean-Themed-Timer


> A calm, minimal timer designed with an ocean-inspired aesthetic.

---

## Overview

Ocean Timer is a simple web-based timer built using HTML, CSS, and JavaScript.  
It focuses on clean design and smooth user interaction, making it a great beginner-friendly project as well as a visually pleasing utility.

---

##  Features

-  Start, Pause, and Reset controls  
-  Clean and minimal UI  
-  Ocean-themed design  
-  Lightweight and fast  
-  Beginner-friendly code
